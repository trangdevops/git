# Information analysis in Git-Repository #
## Search commits ##
```
git log
git reflog
git tag
git shortlog
```

## Search files ##
```
git show
git diff
git grep
git blame
```

## Search errors ##
```
git bisect
```

## Visualazation and statistics ##
```
git shortlog
gitstats
gitk
(GitKraken) & (Git Flow Chart)
```
